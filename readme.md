# Rick & Morty Universe Explorer

## API Utilizada
**Rick and Morty API**
URL: https://rickandmortyapi.com/

## Funcionamiento
La pagina consume la API publica de Rick and Morty para mostrar
personajes en tarjetas dinamicas generadas 100% desde JavaScript.

Permite buscar por nombre (con debounce), filtrar por estado
(Vivo / Muerto / Desconocido) y por genero, ademas de navegar
por paginas.

## Tecnologias
- HTML5 semantico
- CSS3 (Grid, Flexbox, animaciones)
- JavaScript ES6+ (async/await, arrow functions,
  destructuring, map, filter, forEach)
